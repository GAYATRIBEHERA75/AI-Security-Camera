# AI Security Camera

An AI-powered security camera built using Python and OpenCV.

## Features

- Real-time face detection
- Motion detection
- Automatic snapshot capture
- Video recording
- Date and time display
- Security alert when motion is detected

## Technologies Used

- Python
- OpenCV
- NumPy

## How to Run

```bash
py ai_security_camera.py
```
